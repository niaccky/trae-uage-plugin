# Tech Stack

The project is a **Visual Studio Code Extension** built with **TypeScript**.

## Core Technologies
- **Runtime**: Node.js (VS Code Extension Host)
- **Language**: TypeScript (target ES6, CommonJS modules)
- **Framework**: VS Code Extension API

## Dependencies
- **axios**: Used for making HTTP requests to the Trae API.

## Dev Dependencies
- **TypeScript**: Compiler.
- **ESLint**: Linter (`@typescript-eslint/eslint-plugin`, `@typescript-eslint/parser`).
- **@types/vscode**: Type definitions for VS Code API.
- **@types/node**: Type definitions for Node.js.
